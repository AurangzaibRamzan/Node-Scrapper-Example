# Node-Scrapper-Example


Small example of Scrapping using express,node js and cheerio.

## technologies used

-  cheerio
-  express
-  request


## Install NodeModules
  - `yarn install`
  
## Runs the built app in production mode.
  - `yarn start`


 ## Magic happens on port 8081

  - ` http://localhost:8081/scrape`
